Què t’indica Lighthouse sobre el rendiment d’imatges de les teves pàgines? Quins són els principals problemes? Quines recomanacions ofereix?

Lighthouse recomana els formats optims de les imatges que tenim com PNG o JPEG i les recomana canviar per uns formats WebP o AVIF. També recomana abaixar el tamany per les imatges. Codificar les imatges més eficaçment.
